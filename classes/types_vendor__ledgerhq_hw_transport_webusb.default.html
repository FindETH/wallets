Static listen

  • listen(observer: Observer<DescriptorEvent<USBDevice>>): Subscription

  • Overrides default.listen

    • Defined in src/types/vendor/@ledgerhq/hw-transport-webusb.d.ts:27

    Listen to all device events for a given Transport. The method takes an Observer of DescriptorEvent and returns a Subscription (according to Observable paradigm https://github.com/tc39/proposal-observable). Each listen() call will first emit all potential devices already connected and then will emit events that can come over time, for instance if you plug a USB device after listen() or a Bluetooth device becomes discoverable.

    Must be called in the context of a UI click.

    Parameters

    • observer: Observer<DescriptorEvent<USBDevice>>

      The observer object.

    Returns Subscription

    A Subcription object on which you can .unsubscribe(), to stop listening to descriptors.
